# Task 6.1 Plan Request — Client Creation (Top‑Level)

## Goal

Add a simple “New Client” flow on `/clients` so users can create a client with **first + last name only**.

## Requirements

- Add a **New Client** button on `/clients`.
- Open a minimal modal or inline form (first name, last name).
- POST `/clients` to create.
- On success, redirect to the new client detail page.

## Deliverable

- A plan (5–10 steps) covering API + UI changes.
