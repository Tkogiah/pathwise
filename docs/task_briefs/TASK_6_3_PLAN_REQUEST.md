# Task 6.3 Plan Request — Program Metadata (Start Date + Length)

## Goal
Let users edit **program start date** and **program length (days)** per roadmap instance on the client detail page.

## Requirements
- Add inline editable fields for start date + program length (days) in the client detail header area.
- Save/Cancel buttons (no auto‑save).
- Store on **program instance** (not global per client).
- Use these fields later to drive “days in program” gauges.

## Deliverable
- A plan (5–10 steps) covering API changes, UI form state, and validation.
